# 网站结构说明

## 页面结构

### 1. 首页 (Home) - `/`
- **功能**：随机展示废墟图片，横向每行三个
- **交互**：
  - 鼠标悬浮时图片缩放（scale-110）
  - 显示标题和城市信息遮罩
  - 点击图片进入探索页面
- **布局**：响应式网格，移动端1列，平板2列，桌面3列

### 2. 探索 (Exploration) - `/exploration`
- **功能**：瀑布流布局展示所有探索图片
- **筛选器**：
  - 城市筛选（下拉菜单）
  - 类型筛选（工业、学校、住宅、医院、商业、其他）
  - 状态筛选（废弃、已拆除、已翻新、已保护）
  - 年代筛选（按年份）
  - 排序（按时间、城市、类型、状态）
- **布局**：瀑布流，每行三个图片（响应式）
- **交互**：点击图片打开全屏模态框，支持键盘导航

### 3. 在现场 (On-site) - `/onsite`
- **功能**：记录展示，卡片式布局
- **筛选**：
  - 类型筛选（全部、工业、学校、住宅、医院、其他）
  - 排序（按时间、城市、类型）
- **布局**：瀑布流卡片，每行三个
- **交互**：点击卡片进入详细页面

### 4. 装备 (Equipment) - `/equipment`
- **功能**：展示城市探索所需装备
- **分类**：照明、防护、摄影、导航、其他
- **布局**：网格布局，每行4个（响应式）
- **内容**：图标、名称、分类、描述

### 5. 关于 (About) - `/about`
- **内容**：
  - 背景介绍
  - 目标说明
  - 方法原则
  - 联系方式
  - 免责声明

## 设计特点

### 配色方案
- **主色调**：黑色 (#000000)
- **背景**：纯黑 + 深灰卡片 (#171717, #262626)
- **文字**：白色主文字，灰色次要文字
- **边框**：深灰色 (#262626, #404040)

### 响应式设计
- **移动端**：单列布局
- **平板**：2列布局
- **桌面**：3-4列布局

### 动画效果
- **图片悬浮**：缩放效果（scale-110），过渡时间500ms
- **卡片悬浮**：边框颜色变化，图片缩放
- **按钮交互**：颜色过渡，简洁流畅

### 导航栏
- **固定顶部**：始终可见
- **半透明背景**：黑色80%透明度 + 毛玻璃效果
- **当前页面高亮**：白色文字 + 下划线

## 技术实现

### 数据管理
- 使用 TypeScript 类型定义
- 数据文件分离（exploration.ts, onsite.ts, equipment.ts）
- 支持筛选和排序功能

### 组件复用
- `ImageWithFallback`：图片加载失败处理
- `ImageModal`：全屏图片查看
- 响应式布局组件

### 性能优化
- 图片懒加载
- 响应式图片尺寸
- 客户端状态管理（useState, useMemo）

## 使用说明

1. **启动服务器**：
   ```bash
   npm run dev
   ```

2. **访问网站**：
   http://localhost:3000

3. **添加内容**：
   - 编辑 `src/lib/data/` 下的数据文件
   - 添加图片到 `public/images/` 目录

## 注意事项

- 所有图片路径使用 `/images/` 开头
- 确保图片文件存在于 `public/images/` 目录
- 筛选和排序功能使用客户端状态管理
- 响应式设计已优化移动端体验
